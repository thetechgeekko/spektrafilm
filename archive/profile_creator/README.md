Archived research and unsupported experiments from `spektrafilm_profile_creator`.

These files are kept in-repo so useful ideas are not lost, but they are not part of the active profile-creation workflow.

Current archive split:
- `masking_and_sensitivity_experiments.py`: masking-coupler and sensitivity-unmix prototypes.
- `curve_correction_experiments.py`: manual density-curve normalization and gamma-alignment helpers.
- `plotting_experiments.py`: plotting utilities not wired into the active workflow.
- `print_filter_batch_experiments.py`: old batch fitting helper superseded by scripts.
